== Description ==

WP_EmailFeedburnerPop is a plugin based on sexylightbox, with wich you could make your users
suscriptions to your feedburner quickly and easy. Beside this WP_EmailFeedburnerPop works as a
lightbox tool for your images galleries.

**[Download now](http://downloads.wordpress.org/plugin/wp-emailfeedburnerpop.zip)**

[Support](http://puydi.net/blog/wp_emailfeedburnerpop-popup-plugin-para-wordpress/)


== Installation ==

This section describes how to install the plugin and get it working.

1.Download the file WP_EmailFeedburnerPop.rar

2.Unzip WP_EmailFeedburnerPop and upload the folder to the `/wp-content/plugins/` directory.

3.Enter in your admin panel of wordpress and activate the plugin.

4.Join feedburner, select publicize, after select Email Subscriptions.

5.Copy the code of formulary and activate feedburner email subscription.

6.Sign in to the folder WP_EmailFeedburnerPop and open the file `form.html`
paste the feedburner code down line (--Debajo de esta linea pega tu codigo de email de feedburner--)

7.Upload the file `form.html` again.


= spanish Installation =

1.Descargar el archivo WP_EmailFeedburnerPop.rar 

2.Descomprime WP_EmailFeedburnerPop y sube la carpeta a tu servidor en el directorio `/wp-content/plugins/` 

3.Ingresa en tu panel de administracion de wordpress y activa el plugin. 

4.Ingresa a feedburner, y en la seccion publicize selecciona Email Subscriptions. 

5.Copia el codigo del formulario que te dara feedburner y activa la suscripcion por email. 

6.Ingresa en la carpeta WP_EmailFeedburnerPop y abre el archivo `form.html`
pega el codigo de feedburner debajo de la linea (--Debajo de esta linea pega tu codigo de email de feedburner--)

7.Sube de nuevo el archivo `form.html` y remplazalo.
